How it works

1 · Reads

Reads the game's on-screen text from RetroArch's RAM.

2 · Matches

Matches it to a pre-recorded AI voice clip.

3 · Speaks

Plays it live — read-only, and silent on any line it doesn't recognize.

Get set up

  1. RetroArch with the Beetle PSX HW core (PCSX-Redux supported as an alternate).
  2. mpv (free) for playback.
  3. Your own legally-owned copy of Xenogears — no game data is included.

Pairs with Xenogears HD

Stacks cleanly with 1vierock's Xenogears HD texture pack on the same RetroArch + Beetle PSX HW core — textures + voice, all non-invasive. Get their pack from vierockhd.ca. (We don't redistribute their work.)
